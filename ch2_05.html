BMI (進階題)


BMI = weight(KG) / height^2(M^2)
For example, 52 Kg, 155 cm, BMI =52 KG/1.55^2 (M^2) = 21.60

Input:
(1) height (Integer, cm)
(2) weight (Integer, pound 磅 )
(3) -1 (exit)
(4) 1 M = 100 cm, 1 pound = 0.454 KG

Output:
(1) Normal height: 0.5~2.50 M (0.5<=height<=2.50).
   Normal weight: 20~300 KG (20<=weight<=300).
(2) If height not in normal, then output "Input Height Error".
   If weight no in normal, then output "Input Weight error".
(3) If 18.50<=BMI<=24.00,then output the BMI value
   (無條件捨去至小數點後第二位).
(4) If BMI too high (>24.00), then output "BMI too high".
(5) If BMI too low (<18.50), then output "BMI too low".

If else



if
 假如<條件式>成立，就會執行<敘述區段>，否則不執行。 

              if(條件式)
              {
                敘述區段; // 若敘述區段只有一行敘述，大括號可省略
              }

if else
 假如<條件式>成立，就會執行<敘述區段A>，否則就會執行<敘述區段B>。 

              if(條件式)
              {
                敘述區段A;
              }
              else
              {
                敘述區段B;
              }

If ... else  if ... else



一個 if 語句後可跟一個可選的 else if...else 語句，這可用於測試多種條件。

當使用 if...else if...else 語句時，以下三點需要注意：
 1. 一個 if 後可跟零個或一個 else，else 必須在所有 else if 之後。
 2. 一個 if 後可跟零個或多個 else if，else if 必須在 else 之前。
 3. 一旦某個 else if 匹配成功，其他的 else if 或 else 將不會被測試。

              if( x == 10 )
                printf("A, ");

              else if( x == 20 )
                printf("B, ");

              else if( x == 30)
                printf("C, ");

              else
                printf("沒有匹配的值, ");

              printf("x 的值是 %d\n", x);
若 x =  10, 程式結果 : A, x 的值是 10
若 x =  20, 程式結果 : B, x 的值是 20
若 x = 100, 程式結果 : 沒有匹配的值, x 的值是 100

if ... else 搭配運算子



邏輯運算子 || 的意義為「或」。 A || B : 若A、B其中只要有一個為真，結果為真。 

              // 判斷身高是否在範圍內
              if(height_m <= 0.5 || height_m >= 2.50)
              {
                printf("Input Height Error\n");
              }

While Loop



當程式中一個敘述區段需重複執行的次數無法預測時，可使用條件式迴圈來解決。

 1. 前測試迴圈：while...
  若<條件式>成立，則進入迴圈內執行<敘述區段A>一次；
  否則離開迴圈，執行敘述區段B。 

              while(條件式)
              {
                敘述區段A;
              }
              敘述區段B;
 2. 後測試迴圈：do...while
  先執行迴圈內的敘述圈段A一次，再檢查是否滿足<條件式>。
  若<條件式>成立，則繼續執行<敘述區段A>；
  否則離開迴圈，執行<敘述區段B>。 

              do{

                敘述區段A;

              }while(條件式);
              敘述區段B;

While Loop 流程圖





「無窮迴圈」表示迴圈的<條件式>結果無法變為false，一旦執行便是無止盡的迴繞
無法離開迴圈，所以編寫迴圈程式時要特別留意<條件式>。

break




使用時機：主要用於中斷目前的迴圈執行。

以下面程式來說，如果執行到 if 敘述時其<條件式>成立，執行 break 敘述之後，程式流程就會省略<敘述區段B>，直接跳到<敘述區段C>去執行。 

              while ( ..... )
              {
                  敘述區段A;
                  if (條件式)
                  {
                    break ; // 離開迴圈
                  }
                  敘述區段B;
              }
              敘述區段C;
            

              int i = 0;    
              while(i < 10)
              {
                  i++;
                  if(i == 5)
                      break;

                  printf("i = %d, ", i);
              }
output : i = 1, i = 2, i = 3, i = 4,

continue




使用時機：在某些情況下如果需要暫停『本次』迴圈時使用。

以下面程式來說，如果執行到 if 敘述時其<條件式>成立，執行continue敘述之後，程式流程就會省略<敘述區段B>(當作本次迴圈已經執行完畢)，回到迴圈的<條件式>判斷。換言之就是本次迴圈執行完畢，繼續下一次的條件式檢視。 

              while ( ..... )
              {
                敘述區段A;
                if (條件式)
                {
                  continue ; // 暫停『本次』迴圈，回到迴圈的條件式判斷
                }
                敘述區段B;
              }
            

              int i = 0;    
              while(i < 10)
              {
                  i++;
                  if(i == 5)
                      continue;

                  printf("i = %d, ", i);
              }
output : i = 1, i = 2, i = 3, i = 4, i = 6, i = 7, i = 8, i = 9, i = 10,
